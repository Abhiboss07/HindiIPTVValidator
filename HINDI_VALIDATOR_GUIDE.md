## ğŸ” Hindi Channel Detection

The validator identifies Hindi channels using:

1. **Keyword Matching**: Channel names containing Hindi-related keywords
2. **Country Filtering**: Channels from India (IN)
3. **Language Tags**: Explicit [Hindi] tags in channel names

## ğŸ“ Notes

### Demo Mode vs Full Validation
- **Demo Mode** (`hindi_validator_demo.py`): Simulates validation, no ffprobe required
- **Full Validation** (`hindi_validator.py`): Requires ffprobe for actual stream testing

### Channel Sources
The validator works with any M3U file containing Hindi channels. Simply provide your input file and it will:
1. Parse all channels
2. Identify Hindi ones
3. Remove dead/blocked streams
4. Organize by category
5. Save clean playlist

### Customization
- Add more Hindi channels to `channels.yml`
- Modify detection keywords in `is_hindi_channel()` function
- Adjust filtering criteria as needed
